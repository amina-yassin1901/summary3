# Создать приложение для управления задачами, которое синхронизируется с сервером.

Создайте компонент TodoApp
Используйте useState для хранения:
Массива задач
Текста новой задачи
Состояния загрузки
Состояния ошибки
Используйте useEffect для загрузки при старте
Используйте axios для работы с JSONPlaceholder API

API для запросов - https://jsonplaceholder.typicode.com/todos?_limit=5

При первом запуске — загрузить 5 задач с сервера
Добавление задачи — POST запрос на сервер (имитация)
Удаление задачи — DELETE запрос
Переключение статуса — PATCH запрос (выполнено/не выполнено)

Критерии выполнения
При открытии страницы загружаются 5 задач с сервера
Можно добавить новую задачу (она появляется в списке)
Можно отметить задачу выполненной (чекбокс)
Можно удалить задачу (крестик)
Есть индикатор загрузки и обработка ошибок
Счетчик показывает прогресс
Локальное состояние — сохранять новые задачи, даже если API их не сохраняет по-настоящему
